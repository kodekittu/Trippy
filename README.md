# Trippy

Trippy is a small terminal-based travel survey application.
When the app starts, it asks for a user id and name, then shows a simple menu.
Users can view available surveys, start a travel planning survey, and come back later to see their saved survey status.
Completed surveys can show a basic trip itinerary based on the selected country, city, budget, duration, travel style, and activities.
The app also has an analytics section for country stats, city stats, and completion rate.
There is a small admin section to create or list surveys.
Survey data, users, and responses are currently stored in memory while the app is running.
